# Web Development Projects

This repository contains my web development projects showcasing HTML, CSS, and JavaScript skills.

## Projects

### Project 1: Multiplication Table Generator
- **Description**: Interactive web application for generating multiplication tables with customizable parameters
- **Technologies**: HTML, CSS, JavaScript
- **Features**: 
  - User input validation
  - Dynamic table generation
  - Responsive design
  - Customizable table dimensions

### Project 2: Bug Smasher Game
- **Description**: jQuery-based interactive bug smashing game with score tracking and timer
- **Technologies**: jQuery, HTML, CSS, JavaScript
- **Features**:
  - Progressive difficulty levels
  - Score tracking system
  - Timer functionality
  - Collision detection
  - Smooth animations
  - Game rules modal
  - Speed adjustments

## How to Run
1. Clone this repository
2. Open any project's HTML file in a web browser
3. Enjoy the interactive applications!
